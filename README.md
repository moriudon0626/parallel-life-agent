# Parallel Life Agent

対話型AI生態系シミュレーター

## Setup

```bash
npm install
npm run dev
```

## Configuration

1. Open settings (top right)
2. Select AI Provider (OpenAI or Anthropic)
3. Enter your API key
4. Start playing!

## Tech Stack

- React 19 + TypeScript
- Three.js (R3F) + Rapier Physics
- Zustand 5
- OpenAI / Anthropic API
- Tailwind CSS
